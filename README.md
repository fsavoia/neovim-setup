## Pre-requisite

1. Clone this repository
2. Change the name fsavoia for you username

## Setup

1. Open and save the file `~/.config/nvim/lua/fsavoia/plugins-setup.lua`
2. Open neovim and rum `:Mason` and wait until everything is installed
